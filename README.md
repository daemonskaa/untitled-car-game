# untitled-car-game
This is my personal project that I made as some sort of demo for the implementation of _spritestacking_ and other methods to create a pseudo-3D look in 2D.
> built with Godot 4.3

### Tools used
- Godot 4.3 (game engine)
- MagicaVoxel (3D models)
- Aseprite (2D sprites)
